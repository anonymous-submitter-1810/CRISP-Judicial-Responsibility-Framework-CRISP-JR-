# CRISP-JR Framework

This repository contains the visual framework and supplementary materials for the paper:
**"BRIDGING PROCESS MINING AND THE COURTROOM THROUGH CRISP-JUDICIAL RESPONSIBILITY FRAMEWORK (CRISP-JR)"**

## Contents
- `figures/` - All visualizations referenced in the paper
- Significant transition patterns map for Cluster 2 (Figure 1)
- Case duration distribution by cluster, grouped by chamber (Figure 2)
- Activity percolation roles heatmap (Figure 3)
- Duration distribution comparison by chamber within cluster and year (cases started and ended in same year) (Figure 4)

## Code Availability
The implementation code for the CRISP-JR framework will be made available upon paper acceptance to ensure review integrity and proper attribution.
